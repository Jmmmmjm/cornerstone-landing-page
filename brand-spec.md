# Cornerstone · Brand Spec
> Status: Extracted from Offer Document & Codebase

## 🎯 Core Assets
### Logo
- Path: `src/components/Logo.tsx` (Geometric L-shape with Triangle)
- Style: Industrial, System-oriented

### Palette
- **Navy (Midnight):** `#0A192F` (Primary Background)
- **Teal (Neon):** `#64FFDA` (Action/Highlight)
- **Slate (Text):** `#8892B0` (Secondary Content)
- **Light:** `#F8F9FA` (Contrast Base)

### Typography
- **Display:** Bold, tracking-tighter (Headlines)
- **Mono:** HUD style, tracking-widest (Telemetry/Labels)
- **Sans:** High-readability (Body)

## 🎨 Design Philosophy: "Scale On Stability"
- No decorative "fluff". 
- Every element must feel like part of a stable architecture.
- Use of "System Telemetry" (monospaced coordinates and versioning) as a design element.
